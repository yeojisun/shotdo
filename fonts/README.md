# 온글잎 은총체 폰트 파일 업로드 안내

사용자님께서 다운로드받은 **'온글잎 은총'** 폰트 파일을 이 폴더(`public/fonts/`)에 넣어주시면, 웹 서비스 전체의 주간 캘린더 손글씨 폰트가 즉시 반영됩니다.

## 폰트 파일 요구사항
- **지원 파일 확장자**: `.ttf`, `.woff`, `.woff2`
- **적용할 파일명**: 아래 파일명 규칙에 맞춰 폰트 파일 이름을 변경하여 업로드해 주세요.
  * **WOFF2 형식**: `ownglyph_eunchong.woff2`
  * **WOFF 형식**: `ownglyph_eunchong.woff`
  * **TTF 형식**: `ownglyph_eunchong.ttf`

## 폴더 경로
`shotdo/shotdo-web/public/fonts/`

> [!TIP]
> 폰트 파일을 업로드하지 않은 상태에서는 기본 구글 웹폰트인 **'나눔손글씨 펜체(Nanum Pen Script)'**가 대체 폰트로 자동 노출되므로 화면이 깨지지 않고 부드럽게 유지됩니다.
